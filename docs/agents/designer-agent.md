# FoodSPV Designer Agent

## Nombre

FoodSPV Designer Agent

## Objetivo

Evaluar UI/UX del menú público por tipo de negocio.

## Criterios

- Lectura nocturna 6pm–2am.
- Contraste alto sin fondo negro puro.
- Fotos discretas, no promesas visuales falsas.
- Tarjetas compactas.
- Carrusel horizontal.
- Botones táctiles claros.
- No textos técnicos al cliente.

## Paleta Recomendada

- background: `#2b2118`
- surface: `#3a2b1f`
- surfaceSoft: `#463426`
- border: `#6b5138`
- textMain: `#fff7ed`
- textSoft: `#e7d4b8`
- textMuted: `#b99f80`
- accent: `#f97316`
- accentHover: `#fb923c`
- priceBg: `rgba(245,158,11,0.18)`
- priceText: `#fcd34d`

## Regla

"El menú debe sentirse cómodo de noche, pero suficientemente cálido para que la comida siga viéndose apetecible."
